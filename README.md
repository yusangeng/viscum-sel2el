## 综述 | Abstract

DOM element selector.

## 安装 | Install

``` bash
npm install @viscum/sel2el --save
```

## 使用 | Usage

``` ts
import sel2el from '@viscum/sel2el'

// return document.querySelector(input)
const el = sel2el('#foobar')

// return input element
const el2 = sel2el(el)
```
